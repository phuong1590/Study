Combinators:

  1. Adjacent sibling selectors are the selector that allows you to style any specific elements after a specific element.For example:You can style any "a" tag after a "h1" tag...So the first element go first,followed by a simple "+" and then the elements you want to style.

  2. General sibling selectors,the same idea but the elements have to be shared by the same parents.For example:Any "button" you wanted to style after a "textarea" must have the same parents,maybe a "form" tag.

  3. Child selectors will style a direct element as a child of another specified element.For example:You can use the Child selector when you want to style your "li",which is having an "ul"/"ol" as it's parents directly.

  4. Descendant selectors will style any element of any specific element even though they don't have a directly connection.For example:An "ol" contains a "li" and an "ul" that also contains another "li",when you style "li" element while having "ol" as the specific element,both of them will be style.Because eventually,the 2nd "li" was propagated by the "ol".
